# 🏆 머신러닝 컴페티션

다양한 데이터 분석 경진대회 참가 및 수상 프로젝트입니다.

## 📋 프로젝트 개요

### 1. 데이터 전처리
- 데이터 클리닝
- 특성 엔지니어링
- 데이터 증강
- 결측치 처리

### 2. 모델 개발
- 회귀/분류 모델
- 앙상블 기법
- 하이퍼파라미터 튜닝
- 모델 최적화

### 3. 성능 평가
- 교차 검증
- 모델 평가
- 결과 분석
- 앙상블 전략
