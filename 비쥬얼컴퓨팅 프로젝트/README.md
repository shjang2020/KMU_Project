# 🎨 비주얼 컴퓨팅 프로젝트

## 📋 프로젝트 개요

### 1. 3D 그래픽스
- 3D 모델링
- 렌더링
- 애니메이션
- 물리 시뮬레이션

### 2. 컴퓨터 애니메이션
- 캐릭터 애니메이션
- 모션 캡처
- 파티클 시스템
- 물리 기반 애니메이션

### 3. 시각화 기법
- 데이터 시각화
- 인터랙티브 시각화
- VR/AR
- 실시간 렌더링
